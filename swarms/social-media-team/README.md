# Social Media Team Swarm

A 3-agent team optimized for high-quality social media content and engagement.

## Agents

### 1. Trend Spotter (@Spotter)
- **Role**: Scans X (Twitter), LinkedIn, and Reddit for trending topics in AI/Web3.
- **Output**: Daily trend report with hook ideas.

### 2. Viral Ghostwriter (@Ghost)
- **Role**: Transforms trends into high-engagement thread/post drafts.
- **Style**: Optimized for the user's specific voice and platform algorithms.

### 3. Engagement Guard (@Guard)
- **Role**: Drafts replies to comments, monitors sentiment, and identifies high-value accounts for interaction.

## Workflow
1. `@Spotter` identifies 3 trends.
2. `@Ghost` drafts a thread for one trend.
3. `@Guard` reviews the draft for brand safety and suggests hooks.
4. Final draft sent to user for approval via Telegram/WhatsApp.

## Usage
Launch the swarm by sending:
`@Andy launch Social Media Team for "latest GPT-5 rumors"`
